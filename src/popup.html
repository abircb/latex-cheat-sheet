\(\LaTeX\) is a programming language that can be used for writing documents. It is especially useful for the mathematics and sciences fields due to its ease of writing special symbols and equations while also making them look good. Most textbooks are actually written in LaTeX. Due to the programming aspect, writing documents in LaTeX allows a lot more freedom to format how the document should look. However, it is important to learn how these formatting techniques function, and how to deal with LaTeX's unfamiliar syntax

This is why we created this Chrome Extension, so that it be used as a syntax-guide while writing TeX documents using on-line text-editors, like Overleaf or a TeX distribution, like Tex Live

For high-school and university students starting out, LaTeX will make writing homework assignments, publications, posters, and many other documents far easier. For academics and others who already use LaTeX, feel free to reccomend your changes to this cheat sheet on GitHub

LaTeX provides 7 levels of depth for defining sections

Command Comment Level
\part{''part''} Eg: Part 1 of the document, Part 2 of the document, etc. -1
\chapter{''chapter''} Used for books and reports 0
\section{''section''} Not in letters 1
\subsection{''sub''} Eg: 1.1, 1.2, etc. 2
\subsubsection{} Eg: 1.2.1, 1.3.2, etc. 3
\paragraph{''para''} Good for technical specifications 4
\subparagraph{''''} Not in letters, just like levels 1-4 5

When it comes to bibliography management packages, there are three main options in LaTeX: bibtex, natbib, and biblatex. BibLaTeX is a modern program that provides an easier and more flexible interface compared to the other two options. This section explains how to use BibLaTeX to manage and format the bibliography in a LaTeX document.

\usepackage{biblatex} Imports the biblatex package
\addlibresources{references.bib} Imports the bibtex data file that includes information about the references
\cite{''something''} Inserts a reference within the document
\printbibliography Prints the list of cited references

BibLaTeX entry types:

@article{turing,
author = "Alan Turing",
title = "{Computing Machinery and Intelligence}. ({English})",
journal = "Mind",
volume = "LIX",
number = "236",
pages = "433--460",
year = "1950",
DOI = "https://doi.org/10.93/mind/"
} 		A simple example

@misc If nothing else fits
@phdthesis A PhD. thesis
@techreport A technical report/specification

LaTeX documents are plain documents with a .tex extension. But, how do you actually generate the final output?

Answer: It depends on the type of document you want to generate.

There are three output formats available in all TeX distributions. To generate a specific output, the document has to be compiled by running a command in a terminal (Unix-based) or the command prompt (Windows)

latex mydoc.tex This will generate a Device independent file format (DVI document) that consists of binary data
pdflatex mydoc.tex This will create "mydoc.pdf", a Portable Document Format (PDF) document

Of course, the easiest way to compile a TeX file is using an on-line ready to use option like Overleaf, which is what this Chrome extension was made for.
